## 주기능
### 홈 화면
* 영상 리스트를 RecyclerView에 표시합니다.
* 유튜브처럼 화면 하단에 재생된 영상을 미리보기 할 수 있으며, 재생/일시정지 버튼을 눌러 영상을 재생하거나 일시정지 할 수 있습니다.
* 영상을 클릭하면 화면 하단의 작은 미리보기 화면이 커지며 영상을 감상할 수 있습니다.
